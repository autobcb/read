##阅读是什么?
一个全移动端可用的阅读app，这个仓库是阅读app服务端源码，阅读app可以在全端同步进度。

##cookie
有些书源有key验证，这个验证是通过java.getCookie接口获取的这个接口获取的参数可在后台添加。

##推荐书源
*[晴天](http://api618.qingtian618.site) (需要修改书源地址，再后台添加cookie后使用)

##暂不支持
阅读不支持书源链接带有\n，不支持登陆
暂不支持文本以外的书源
